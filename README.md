# Example Airbnb

A simple Airbnb-style UI demo built with modern React tooling.

## Live Demo
https://example-airbnb.vercel.app/

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Run Locally
Requirements: Node.js + npm (or bun)

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev

## Build
npm run build
npm run preview

## Deployment
Deploy with Vercel: import this repository, then Vercel will build and publish automatically on every push to the main branch.

## Custom Domain
In Vercel: Project → Settings → Domains → add your domain and follow the DNS instructions.
